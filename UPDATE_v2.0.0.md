## 📦 Docker 镜像信息

### 镜像标签
- `jinkang19940922/gycsj-image:latest` - 最新稳定版
- `jinkang19940922/gycsj-image:2.0.0` - v2.0.0 版本

### 拉取镜像
```bash
# 拉取最新版本
docker pull jinkang19940922/gycsj-image:latest

# 拉取指定版本
docker pull jinkang19940922/gycsj-image:2.0.0
```

### 运行容器
```bash
docker run -d \
  -p 23400:3000 \
  -v $(pwd)/uploads:/app/uploads \
  -v $(pwd)/thumbnails:/app/thumbnails \
  -v $(pwd)/data:/app/data \
  -e ADMIN_USERNAME=admin \
  -e ADMIN_PASSWORD=admin123 \
  --name gycsj-image \
  jinkang19940922/gycsj-image:latest
```

### 使用 Docker Compose
```bash
# 启动服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

## 📖 使用说明

1. **访问应用**：打开浏览器访问 `http://localhost:23400`
2. **登录系统**：使用默认账号（admin / admin123）登录
3. **查看登录界面**：欣赏必应每日壁纸和每日一言
4. **图片库**：查看和管理已上传的图片
5. **公共图片**：浏览必应壁纸，支持下载和转存
6. **图片预览**：点击图片进入预览模式，支持放大缩小、全屏、拖拽
7. **API工具**：生成随机图片API链接，支持配置保存

## 📂 目录结构

```
gycsj-image/
├── public/              # 前端资源
│   ├── index.html       # 主页面
│   ├── css/             # 样式文件
│   └── js/              # 脚本文件
├── src/                 # 后端代码
│   ├── controllers/     # 控制器
│   ├── services/        # 服务层
│   ├── middleware/      # 中间件
│   └── routes/          # 路由
├── uploads/             # 上传的图片
├── thumbnails/          # 缩略图
├── data/                # 数据文件
├── server.js            # 服务器入口
├── Dockerfile           # Docker 构建文件
└── docker-compose.yml   # Docker Compose 配置
```
