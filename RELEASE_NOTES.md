# v2.2.0 版本发布说明

## 🎉 版本概述

**光影穿梭机 HOME v2.2.0** 正式发布！本次更新带来了全新的展厅模式功能，优化了图片预览体验，并修复了多个已知问题。我们致力于为用户提供更加流畅、沉浸式的图片管理和浏览体验。

---

## ✨ 核心亮点

### 🖼️ 全新展厅模式

体验前所未有的沉浸式图片浏览！展厅模式将您的图片库转化为动态画廊：

- **4列动态展示**：图片以流动动画形式呈现，营造艺术画廊氛围
- **双向流动效果**：上下交替流动，视觉体验更丰富
- **全屏沉浸体验**：隐藏所有界面元素，专注于图片本身
- **智能交互设计**：鼠标移动显示控制按钮，静止自动隐藏
- **快捷键支持**：ESC键一键退出，操作更便捷

### 🔄 预览功能重构

简化操作流程，提升使用效率：

- **移除冗余切换**：点击图片直接预览原图，无需额外切换
- **界面更加简洁**：去除原图/缩略图切换按钮，视觉更清爽
- **操作更加直观**：减少用户操作步骤，提升使用效率

### 🖥️ 全屏显示优化

修复全屏模式下的显示问题：

- **消除黑边问题**：图片完整填充屏幕，无多余边框
- **优化显示比例**：确保图片以最佳比例展示
- **提升视觉体验**：全屏浏览更加沉浸

## 🚀 快速开始

### 升级方式

**Docker 用户：**
```bash
# 拉取最新镜像
docker pull jinkang19940922/gycsj-image:latest

# 重启容器
docker restart gycsj-image
```

**Docker Compose 用户：**
```bash
# 拉取并重启
docker-compose pull
docker-compose up -d
```

**源码部署用户：**
```bash
# 拉取最新代码
git pull origin main

# 重启服务
npm restart
```

### 使用展厅模式

1. 登录系统，进入"图片库"页面
2. 点击工具栏中的"展厅模式"按钮
3. 享受沉浸式图片浏览体验
4. 按 ESC 键或点击"退出展厅"按钮退出

## 🔧 技术细节

### 主要修改文件

- `public/index.html` - 添加展厅模式样式和按钮
- `public/js/main.js` - 实现展厅模式功能和交互优化

### 新增功能模块

- 展厅模式管理：`toggleGalleryMode()`, `exitGalleryMode()`
- 列复制与恢复：`duplicateColumnsForShowroom()`, `restoreColumnsFromShowroom()`
- 鼠标检测系统：`startShowroomMouseDetection()`, `handleShowroomMouseMove()`
- 键盘快捷键：ESC键退出展厅模式

### 性能优化

- 使用 CSS 动画和 GPU 加速
- 优化图片加载和渲染性能
- 减少不必要的 DOM 操作
